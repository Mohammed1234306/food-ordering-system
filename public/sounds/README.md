# Alert Sound Files

Place your MP3 alert sound files in this directory:
- bell1.mp3  → "Classic Bell" sound
- bell2.mp3  → "Chime" sound  
- bell3.mp3  → "Alert" sound

## Where to get free notification sounds:
- https://freesound.org (search "notification bell")
- https://mixkit.co/free-sound-effects/notification/
- https://notificationsounds.com

## Format requirements:
- Format: MP3 or OGG
- Duration: 1-3 seconds
- File size: Keep under 100KB for fast loading

## Using external URLs instead:
In the admin Settings page, you can paste any public MP3 URL
in the "Custom sound URL" field to use sounds hosted elsewhere.
